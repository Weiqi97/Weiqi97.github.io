WHAT I'M WORKING ON

Machine learning on encrypted data with Functional Encryption
While traditionally, learning on encrypted data is achieved by utilizing homomorphic encryption schemes, we have noticed recent research on Cryptography has led to the development of flexible encryption schemes that allow for function operation on encrypted data. In this work, we implement regression models that can train on encrypted data with a relatively new cryptographic primitive, functional encryption for inner-product. In this setting, data can be encrypted at the source using a functional encryption scheme and then aggregated at a data server for learning without the data being revealed to other sources.
Implementation of inner-product Function-revealing encryption
The new primitive function-revealing encryption (FRE) allows third parties to grab some ciphertexts and evaluate a pre-defined functionality on their plaintexts with a public oracle, regardless of whether they participated in the encryption scheme. This project implements a multi-client FRE with the inner-product functionality, where we assume each ciphertext provider would be a different client.
